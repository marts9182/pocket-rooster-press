# Privacy Policy — Pocket Rooster Press

**Effective date:** 2026-05-18
**Operator:** Pocket Rooster Press (sole proprietorship, United States)
**Contact:** marts9182@gmail.com

## 1. Scope

This privacy policy covers the internal automation tool used by **Pocket Rooster Press** to publish marketing pins to its own Pinterest business account. The tool is operated solely by Pocket Rooster Press and is not offered to, installed by, or made available to any other person or organization.

## 2. Information We Collect

We collect **no personal information from any third party**, including but not limited to Pinterest users who view, save, click, or otherwise interact with our pins. Specifically, the tool does not collect, request, store, share, or process:

- Names, usernames, email addresses, or contact details of Pinterest users
- Pinterest user IDs, follower lists, or social graph data
- IP addresses, device identifiers, or location data of pin viewers
- Comments, messages, or any user-generated content posted by other accounts
- Analytics or behavioral data tied to identifiable individuals

The only data the tool stores locally on the operator's own computer is:

- An OAuth access token issued by Pinterest to the Pocket Rooster Press business account itself (used to authenticate API requests)
- The operator's own Pinterest board identifiers and pin identifiers (used to track which pins have already been posted to avoid duplicates)
- Aggregated, anonymous performance counters (e.g., "12 pins posted today") from Pinterest's own analytics API, where available

None of the above constitutes personal information of any third party.

## 3. How We Use Information

The OAuth access token described above is used exclusively to authenticate Pocket Rooster Press's own automation tool with the Pinterest API for the purpose of creating pins on the Pocket Rooster Press business account. It is never used to access any other Pinterest account or user data.

## 4. Information Sharing

Pocket Rooster Press does not sell, rent, share, transfer, or disclose any data — its own or otherwise — to any third party. The OAuth token is stored only on the operator's local machine in an encrypted file and is not transmitted to any service other than Pinterest's own API endpoints.

## 5. Data Retention

The OAuth token is retained only as long as needed for the tool to function and is automatically refreshed or discarded when it expires. Local pin history is retained indefinitely on the operator's machine for record-keeping purposes and is never transmitted off-device.

## 6. Children's Privacy

The Pocket Rooster Press automation tool is not directed at children, does not interact with children, and does not collect data from any user of any age.

## 7. International Users

Pocket Rooster Press is based in the United States. The automation tool does not process personal data of users located in the European Union, the United Kingdom, or any other jurisdiction, because it does not process personal data of any third party at all.

## 8. Security

The OAuth access token issued to Pocket Rooster Press is stored locally on the operator's personal computer behind operating-system-level access controls. No data is transmitted to any server other than Pinterest's official API endpoints over HTTPS.

## 9. Changes to This Policy

Pocket Rooster Press may update this policy from time to time. Material changes will be reflected in the "Effective date" above.
